네이버 로그인API 검수 문제로 로그인이 불가하여 주요서비스인 보관함 열람이 불가합니다😂

### Web 실행방법 🐈
```
git clone https://github.com/boostcamp-2020/Project01-C-User-Event-Collector.git
npm i
cd backend
npm i
npm run start
cd ../frontend
npm i

--(develop mode)--
npm run dev 

--(production mode)--
npm run build 
npm run start 
```
